#  EdTech Startup Data Analysis using Power BI

##  Project Overview
This project analyzes online course data for an EdTech startup using **Power BI** to extract actionable insights for business growth. The analysis focuses on understanding course demand, viewer engagement, and in-demand skills through interactive dashboards.

The project is based on a hands-on tutorial by **CampusX** and follows a real-world data analytics workflow.

---

##  Objectives
- Clean and preprocess raw course data
- Analyze course trends across categories, languages, and course types
- Identify high-engagement areas using average views
- Discover in-demand skills through visual analytics
- Build an interactive Power BI dashboard for decision-making

---

##  Tools & Technologies
- Power BI  
- Power Query Editor  
- DAX (basic measures)  
- M Language  
- Custom Visuals (Word Cloud)

---

##  Dataset Description
The dataset contains information related to online recorded courses, including:
- Course ID & Title  
- Category & Sub-category  
- Course Type  
- Language & Subtitle Language  
- Skills Covered  
- Instructor  
- Rating  
- Number of Viewers  
- Duration  
- Platform  

---

##  Data Cleaning & Transformation
The following steps were performed using **Power Query**:
- Removed duplicate, null, and error rows  
- Checked column quality (valid, error, empty values)  
- Extracted numerical ratings from text (e.g., *"4.9 stars"*)  
- Standardized categorical columns  

---

##  Dashboard Features & Insights

### 1️ Course Type Distribution
- Visualized course type distribution across categories using a stacked bar chart
- Helped identify the most popular course formats

### 2️ Average Views Analysis (Drill-Down)
- Analyzed average views by:
  - Category  
  - Sub-category  
  - Language  
- Implemented drill-down hierarchy for deeper insights

### 3️ In-Demand Skills Analysis
- Used a **Word Cloud** custom visual to identify the most in-demand skills
- Skill popularity represented by word size

###  Interactive Filtering
- Category-based slicers for dynamic dashboard interaction

---

##  Key Insights
- Certain categories and languages show higher viewer engagement
- Specific course types dominate high-demand categories
- Skill frequency analysis reveals market-relevant topics

---
